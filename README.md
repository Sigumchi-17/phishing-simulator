# phishing-simulator

본 프로젝트는 **실제 보이스피싱 대화 흐름을 체험하고, 사용자 대응 행동을 기반으로 위험도를 평가하는 시뮬레이터**입니다.
사용자는 다양한 사칭 시나리오(택배, 검찰, 보험, 가족, 연인 등) 속에서 대화를 진행하며,
대화 종료 시 **규칙 기반 평가 시스템**을 통해 자신의 대응 수준에 대한 피드백을 받습니다.

---
## 프로젝트 개요

- **목적**
  - 보이스피싱 상황을 직접 체험하며 위험 행동을 인지
  - 사용자 대응 패턴을 분석하여 실질적인 예방 학습 제공

- **주요 기능**
  - 시나리오 기반 보이스피싱 대화 시뮬레이션
  - 사용자 발화 분석 및 이벤트 감지
  - 규칙 기반 점수 산출 및 위험도 평가
  - 시나리오별 맞춤 피드백 제공

---

## 주요 시나리오

- 택배 사칭
- 검찰 사칭
- 보험사 사칭
- 가족 사칭
- 연인/감정 유도형 사기

---

## 기술 스택

- **Frontend**
  - HTML, CSS, JavaScript
 
- **Backend**
  - Node.js (Express)
  - SQlite
  - OpenAI API (대화 시뮬레이션)

- **평가 시스템**
  - 규칙 기반 이벤트 감지 (JSON Rule Set)
  - 행동별 가중치 점수 누적 방식

---

## 폴더 구조
- backend/ : Express + OpenAI API 서버
- frontend/ : 정적 웹 프론트엔드

---

## 실행 방법 (로컬)
1. backend 폴더에 `.env` 파일 생성
```env
OPENAI_API_KEY=본인_API_키
```
2. backend 실행
```bash
cd backend
npm install
node server.js
```
3. frontend 실행
   `frontend/index.html` 파일을 브라우저로 직접 실행

---

## 기대 효과
- 보이스피싱 수법에 대한 **체험 기반 학습**
- 단순 정보 제공이 아닌 **행동 중심 예방 교육**
- 실제 상황에서의 대응 판단력 향상

---

## 보안 안내
- OpenAI API Key 및 환경 변수는 `.env` 파일로 관리되며 GitHub에 포함되지 않습니다.
- 개인정보 및 실제 금융 정보는 저장되지 않습니다.
